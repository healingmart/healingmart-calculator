HealingMart Calculator v4 Clean Start

1. 기존 healingmart-calculator 저장소 자체는 삭제하지 마세요.
2. 저장소 안의 기존 파일만 모두 삭제한 뒤 이 ZIP의 내용을 저장소 루트에 업로드하세요.
3. 업로드 후 저장소 최상단에 반드시 아래가 보여야 합니다.
   - .nojekyll
   - index.html
   - dist/
4. GitHub Settings > Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
5. 저장 후 배포가 끝나면 아래 주소를 확인하세요.
   https://healingmart.github.io/healingmart-calculator/
6. 진단 페이지에서 모든 항목이 '정상'이면 Blogger HTML을 적용하세요.
